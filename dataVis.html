Data Visualisation

This data visualisation was created using Chart.js. Chart.js is a a charting library which uses HTML5 and JavaScript to create interactive graphs.

Mouseover the bars and points of the line charts to see the data at different years.

By clicking on the categories in the key you can remove the data from the data visual. The chart will automatically update with the removed category(s).

The data used to create the data visualisation can be found on the Data Goverment website.
